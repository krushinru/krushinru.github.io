## About

POAPI is a Russian-language catalog of public APIs from Russian and international companies. It brings together, in one place, the interfaces that can be plugged into a product: payments, logistics, telephony, analytics, machine learning, and dozens of other use cases.

Before POAPI, developers had to hunt for APIs by hand — across company sites and scattered documentation. The catalog shortens that path: open the relevant category and see right away which platforms already offer an API for the task.

### Platform features

**Instant search**

Search works across names, descriptions, and tags. The right API surfaces in a couple of seconds, with no jumping between sections or filters.

**Categories by use case**

23 categories cover the main integration areas: finance, commerce, government services, HR, mailing, geography, telephony, security, and more. The structure helps narrow the choice down to the most relevant platforms.

**Everything in one card**

Each API card gathers what is needed for an initial assessment: a short description, the owning company, categories, and a link to the documentation. That is usually enough to decide whether the solution fits the stack and the business case — before signing up.

**Companies section**

A separate tab lists every company along with all of its APIs. Convenient when you need to see the full integration stack of a single ecosystem — for example, a bank or a marketplace.

### Results

The catalog now lists more than 480 APIs from 135 companies across 23 categories. POAPI is useful to developers, product managers, integrators, and agencies — anyone choosing an API provider for a new task and looking to save a day or two on manual research.
