## О проекте

POAPI — русскоязычный каталог публичных API российских и международных компаний. Сервис собирает в одном месте интерфейсы, которые можно подключить к продукту: платежи, логистику, телефонию, аналитику, машинное обучение и десятки других сценариев.

До POAPI разработчику приходилось искать API вручную — по сайтам компаний и обрывкам в документации. Каталог сокращает этот путь: открываешь нужную категорию и сразу видишь, у кого есть готовое решение под задачу.

### Возможности платформы

**Мгновенный поиск**  
Поиск работает по названиям, описаниям и тегам. Нужный API находится за пару секунд, без перехода между разделами и фильтрами.

**Категоризация по направлениям**  
23 категории охватывают основные сферы интеграций: финансы, торговля, госуслуги, HR, рассылки, география, телефония, безопасность и другие. Структура помогает быстро сузить выбор до релевантных платформ.

**Информация в одной карточке**  
В карточке API собрано всё, что нужно для предварительной оценки: краткое описание, компания-владелец, категории и ссылка на документацию. Этого хватает, чтобы понять, подходит ли решение под стек и бизнес-сценарий, ещё до регистрации.

**Раздел компаний**  
Отдельная вкладка показывает каждую компанию и все её API в одном месте. Удобно, когда нужно посмотреть весь стек интеграций одной экосистемы — например, банка или маркетплейса.

### Результаты

В каталоге собрано более 480 API от 135 компаний в 23 категориях. POAPI пригождается разработчикам, продактам, интеграторам и агентствам — всем, кто выбирает поставщика API под новую задачу и хочет сэкономить день-два на ручном поиске.
